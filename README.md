# 使用wxpy爬取需求信息发送微信

## 技术栈：

python3.6 + pip3 + wxpy + wechat_sender + urllib + bs4 +pymysql

## 下载 (运行`python3.6 `)
##### 如果安装报错，请用pip安装指定模块，对于的wxpy的学习，请github自行搜索

 	git clone https://github.com/Topthinking/wxPython.git
	
 	cd wxPython

 	python main.py
